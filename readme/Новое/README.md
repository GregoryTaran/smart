## 🚀 Описание проекта

**SMART VISION** — это веб-платформа, где мысль превращается в визию,  
визия — в дорогу,  
а дорога — в источник внутреннего света.

---

## 🔐 SmartID — архитектура аутентификации

Платформа использует собственную систему авторизации **SmartID** —  
лёгкую, чистую и стабильную модель, основанную на:

- одной серверной session (HttpOnly cookie),
- одном запросе:
  GET /api/auth/session
- одном объекте `session`, который получает каждая страница,
- отсутствии JWT, LocalStorage, кэшей и событий.

SmartID обеспечивает детерминированное состояние на всех страницах.
