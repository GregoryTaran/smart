НА СЕРВЕРЕ 
voicerecorder.py

ХРАНЕНИЕ ДАННЫХ
Супабейз сторадж.


Основной функционал: 
- Взять user_id у svid
- Записать разговор юзера
- Передать на сервер по частям во время разговора
- По окончании разговора сформировать файл.
- Записать на в сторадж. 
- Вывести результат на страницу ниже, рядом с другими записями в колонку.
- У каждой записи есть: имя записи, дата, id самой записи, файл записи, другое..
- По каждой сохраненной записи юзер может:  Прослушать, Удалить, Переименовать название, Загрузить. 
- На начальном этапе предлагаю хранить на сервер в МП3. 
- Логика такая: передали wav, склеили, перевернули в mp3, сохранили, отдали для вывода информацию и ссылку на файл.
- Логика кнопок при записи уже построена в файле.
